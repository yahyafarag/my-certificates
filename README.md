# Yahya Tarek - Engineering Portfolio

A modern, responsive, and enterprise-grade portfolio website built for Yahya Tarek, showcasing expertise in Factory Equipment, Production Lines, and CMMS Implementation.

## Features
- **Bilingual (English/Arabic):** Full RTL support with `hreflang` meta tags.
- **Progressive Web App (PWA):** Offline capability, installable on mobile devices, powered by `manifest.json` and a Service Worker.
- **Enterprise SEO:** Canonical tags, OpenGraph for social sharing, schema.org JSON-LD, dynamically generated `sitemap.xml`, and optimized meta descriptions.
- **Performance Optimized:** 100% WebP image delivery, lazy loading, and fully responsive CSS Flex/Grid layout.
- **Analytics & GDPR Compliant:** Integrated Google Analytics (GA4) placeholder with a Cookie Consent Banner and Privacy Policy.

## Architecture
- Static HTML5 / CSS3 / Vanilla JavaScript.
- No heavy frameworks used, maximizing load speed and TTFB (Time to First Byte).
- Hosted on GitHub Pages.

## Maintenance
To update content, edit the `.html` files directly. To update styles, edit `assets/css/style.css`.
